# Assets Directory

## Default Bikash QR Code Image

Place your default Bikash QR code image here with the filename:
- `default-bikash-qr.png` (or `.jpg`, `.jpeg`, `.webp`)

This image will be displayed when no `bikashQRCodeId` is set in the database settings.

### Image Requirements:
- Recommended size: 220x220 pixels or larger (square format)
- Format: PNG, JPG, or WebP
- The image should be a QR code that can be scanned for Bikash payments
